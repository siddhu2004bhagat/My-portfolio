# Sudhanshu Kr. Jaiswal - Portfolio Website 🚀

This is my personal portfolio website showcasing my work in Embedded Systems, IoT Development, and Industrial Automation.

## About Me 👨‍💻

Electronics & Communication Engineering student at Lovely Professional University with hands-on experience in:
- Embedded Systems Development
- IoT Solutions & Cloud Integration
- Industrial Automation (PLC Programming)
- Computer Vision & AI
- Sensor Integration & Wireless Communication

## Tech Stack 💻

**Frontend:** React, TypeScript, HTML5, CSS3, JavaScript  
**3D Graphics:** Three.js, WebGL, React Three Fiber  
**Animation:** GSAP (GreenSock Animation Platform)  
**Build Tool:** Vite  
**Languages:** C++, Python, C, Verilog, System Verilog  
**Hardware:** Arduino, ESP32, Raspberry Pi, STM32  
**Tools:** Firebase, MATLAB, Simulink, PLC Programming

## Featured Projects 🔧

1. **IoT Heart & ECG Monitor** - Real-time health monitoring using ESP32 and ThingSpeak
2. **Finger Gesture LED Control** - Computer vision-based gesture recognition with OpenCV
3. **14-Channel RC System** - Wireless communication system using NRF24L01
4. **AI Exam Monitoring System** - Face recognition and behavior detection with Raspberry Pi
5. **ESP32 Thermal Printer** - UART communication interface for thermal printing

## Installation & Setup 🛠️

```bash
# Clone the repository
git clone https://github.com/siddhu2004bhagat/portfolio.git

# Navigate to project directory
cd Siddhu-protfolio

# Install dependencies
npm install

# Run development server
npm run dev
```
